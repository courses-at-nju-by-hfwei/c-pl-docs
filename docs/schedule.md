| 课时 | 内容 |
| ----- | ----- |
| 1 | Introducing C |
| 2 | Variables, Types (int, double), I/O |
| 3 | If, For, Array |
| 4 | For, While, Do-While |
| 5 | Loops (More Examples; break, continue); Multidimensional Arrays |
| 6 | Function |
| 7 | Recursion |
| 8 | Recursion; Data Types |
| 9 | Pointers |
| 10 | Pointers (More) |
